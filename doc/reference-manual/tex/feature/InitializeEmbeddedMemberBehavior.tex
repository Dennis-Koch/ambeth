\SetAPI{J-C}
\section{InitializeEmbeddedMemberBehavior}
\label{feature:InitializeEmbeddedMemberBehavior}
\ClearAPI
Initializes eagerly all embedded properties on an entity type at runtime with an instance of the enhanced property type generated by the \type{EmbeddedTypeBehavior}. This way there will never occur a NullPointerException.
